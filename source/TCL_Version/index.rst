BraineryWiz TCL Version Documentation
====================================================
TCL **BraineryWiz** is package for plotting OpenSees models and also finite element models and is basically is independent of the element. In this document you will find out the way of installation and usage of BraineryWiz TCL package for plotting OpenSees models that are provided using TCL coding.
BraineryWiz is using an element independent approach that do not need to know the type of elements to plot them. So, it makes this package more powerful to plot any type of element without any need of defining them. Also, this package uses GPU to create more fluent and faster. This package is under develop and users using following addresses can find out the latest updates and new features.
