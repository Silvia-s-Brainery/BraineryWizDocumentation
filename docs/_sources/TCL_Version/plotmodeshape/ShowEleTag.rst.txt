ShowEleTag Option
====================================================
By calling this option, tag of elements will be shown on plot.

Example
--------

.. code-block:: tcl
   :caption: Example with ShowEleTag for ModeNumber 3
   
   source BraineryWiz.tcl
   
   # ...
   # lines of Creating OpenSees model
   # ...
   
   # Call PlotModeShape command 
   PlotModeShape 3 ShowEleTag

Watch resulted plot :ref:`here <plotmodelshowelementstagpy>`.
